# burnrate-site

Static page hosting for **BurnRate** — privacy policy, support, and a
small landing.

Live at: `https://<your-github-username>.github.io/burnrate-site/`

## Files

- `index.html` — landing page (visiting card with links)
- `privacy-policy.html` — Privacy Policy (linked from App Store / Play Store)
- `support.html` — Support page (linked from App Store / Play Store)

## Maintenance

Edit the HTML files in this repo, commit, push. GitHub Pages picks up
the change in ~30 seconds.

The privacy policy's "Last updated" placeholder lives in the HTML —
search for `<!-- DATE_HERE -->` and fill in the date when you edit.
